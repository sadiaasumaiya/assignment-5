Important FAQ

1. What is the Document Object Model (DOM)?

The DOM is a programming interface for web documents. It represents the structure of a document as a tree of objects.

2. How do you select an element from the DOM?

You can select an element using methods like getElementById, querySelector, or getElementsByClassName.

3. What is event delegation in the context of the DOM, and why is it useful?

Event delegation is a technique that allows you to add a single event listener to a parent element to manage events for multiple child elements. Itâ€™s useful for improving performance and managing dynamically added elements.

4. How do you manipulate an element's attributes and styles using the DOM?

You can use properties like element.style to change styles and element.setAttribute to change attributes.
